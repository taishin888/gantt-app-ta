設定・連携

GitHub 設定

ℹ️ リポジトリは自動設定済みです。Personal Access Token だけ入力して「設定保存」してください。
Token は このブラウザにのみ保存されます。別のPCでは再入力が必要です。

💡 別PCでも使えるようにするには「☁️ 全PCに反映」をクリックしてください。
トークンがGitHubの docs/index.html に書き込まれ、どの端末でも自動接続されます。

💡 Tokenの取得: GitHub → Settings → Developer settings → Personal access tokens → Generate new token（repoスコープ必須）

Teams Webhook

通知はGitHub Actions経由で送信されます。このURLをGitHub Secretsの TEAMS_WEBHOOK に登録してください。
